## NOTE GENERALI CROSS-REFERENCE

1. **Testate appoggio** → selezione per: materiale corpo (PA o inox) → diametro tubo → footprint piastra
2. **Morsetti portaguide** → selezione per: tipo guida (trapezoidale/tonda/rettangolare) → sistema fissaggio (piatto 25x8, perno, tubo) → diametro perno
3. **Guide prodotto** → selezione per: geometria sezione (mm×mm) → materiale PE → spessore profilo metallico (1,5 o 1,9 mm) → lunghezza (3 o 6 m)
4. **Anelli di fermo** → selezione per: forma foro (tondo/quadro) → diametro/lato → con o senza sede linguetta
5. **Codici Rexnord** = Part. (es. S0237) + Code (es. 63151) → ordine = "S023763151"
